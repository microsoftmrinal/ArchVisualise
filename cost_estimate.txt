Key Observations
----------------
1. Copilot Studio license ($200/mo) is the fixed baseline cost regardless of usage.
2. Azure OpenAI is the primary variable cost — each diagram request uses ~1K input
   tokens and ~2K output tokens for the GPT-4o parsing call. At high volumes this dominates.
3. Container Apps on Consumption plan is very cost-efficient — it scales to zero and
   has a generous free grant. Even at 2,000 requests/day it stays under $30/mo.
4. Storage is negligible — diagram files are small (PNG + .drawio).


Cost Optimization Tips
----------------------
- Switch to gpt-4o-mini for the parsing step to cut OpenAI costs by ~90%
  (input: $0.15/1M, output: $1.25/1M) — architecture JSON extraction doesn't
  require full gpt-4o reasoning.
- Set a TPM/RPM limit on the OpenAI deployment to cap spending.
- Container Apps min-replicas: 0 is already configured — cold starts add ~2-5s
  latency but save cost when idle.
- Consider Azure API Management (Consumption tier, ~$3.50/10K calls) if you need
  rate limiting or external auth.


Realistic Production Estimate
-----------------------------
$220 - $420/month for a moderate internal tool workload.
